# Mouse Jump
Spawn a preview of the virtual screen and click on it to jump to that position.
> Works similar to PowerToys' version except without the customization.


## Requirements
- AutoHotkey v2.1
-  GDI+ Library [Gdip_All.ahk](https://github.com/buliasz/AHKv2-Gdip/blob/master/Gdip_All.ahk)

## How to use
Create a hotkey `$F1::MouseJump()`

When the preview is shown, clicking outside the preview or pressing `Escape` will close it.
